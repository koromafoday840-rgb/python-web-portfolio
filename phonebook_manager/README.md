# 📞 Phonebook Manager (Python)

## 🎯 Project Goal
A command-line interface (CLI) application designed to manage personal contacts using Python's dictionary-based storage.

## 💡 Technical Implementation
* **Memory Management:** Uses a local dictionary to store key-value pairs (Name: Phone Number).
* **Search Optimization:** Implements O(1) time complexity for contact retrieval.
* **CRUD Operations:**
    * **Create:** Add new contacts.
    * **Read:** Search and display existing contacts.
    * **Delete:** Remove outdated records safely.
