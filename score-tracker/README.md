# 📈 Student Score Tracker

## 🎯 Project Goal
A Python utility designed to process student academic data, calculate averages, and assign performance grades dynamically.

## 💡 Technical Implementation
* **Logic Flow:** Uses conditional branching (if/elif/else) to determine grade thresholds.
* **Data Handling:** Processes lists of student scores and performs arithmetic operations for class statistics.
* **Trace Verified:** This logic has been manually traced to ensure 100% accuracy in grade boundary calculations.
